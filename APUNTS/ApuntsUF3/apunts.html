Seleccionar elements DOM

Hi ha la manera antiga i la nova:
La antiga:

  • Per tag: document.getElementsByTagName("li")
  • Per classe: document.getElementsByClassName("card")
  • Per atribut name: document.getElementsByName("nom")
  • Per id: document.getElementById("menu")

La nova:

  • document.querySelector(".cards"): Aquest agafa el 1r element que tingui la classe cards
  • document.querySelectorAll("a"): Aquest agafa TOTS els elements que tinguin el tag "a"

Ara farem els fragments que a diferenca dels altres no formen part del DOM

El createElement crea l'element literal, mentre que el fragment crea un contenedor temporal on s'inserten els elements, cuan s'inserta el fragment, el fragment s'elimina i queda lo de dintre.
Pero aquestes son les versions antigues la llista seguent son les noves:

  • insertAdjacentElement(posicio, element) substitueix a (appendChild)
  • insertAdjacentHTML(posicio, element) substitueix a (innerHTML)
  • insertAdjacentText(posicio, text) substitueix a (textContent)

Les posicions poden ser:

  • beforebegin(germa anterio)
  • afterbegin(primer fill)
  • beforeend(darrer fill)
  • aftered(germa seguent)
